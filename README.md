# Diabetes-Analytics-Dashboard-PowerBI
Interactive Diabetes Analytics Dashboard built using Power BI with data cleaning, Power Query, DAX, and healthcare insights.

## 📌 Project Overview

This project was completed as part of my AI/ML Internship at InternPe. Although the internship focuses on Artificial Intelligence and Machine Learning concepts, I completed this assignment from a Data Analytics perspective by cleaning the dataset and developing an interactive dashboard using Power BI.

The objective was to transform raw healthcare data into meaningful insights through data cleaning, data modeling, and visualization.

## 🎯 Objectives

- Clean and prepare healthcare data
- Create an interactive Power BI dashboard
- Build KPIs using DAX
- Visualize diabetes-related insights
- Improve decision-making through analytics

## 📊 Dashboard Features

- KPI Cards
- Diabetes Outcome Analysis
- Age Group Analysis
- BMI Category Analysis
- Glucose Analysis
- Interactive Slicers
- Professional Healthcare Theme

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel
- Data Cleaning
- Data Visualization

## 📈 Key Insights

- Distribution of diabetic and non-diabetic patients
- Average glucose and BMI comparison
- Age group analysis
- Healthcare trend visualization

## 📌 Note

This project was completed as part of an AI/ML internship. Rather than building a machine learning model, I focused on applying Data Analytics techniques through data cleaning, transformation, and dashboard development using Power BI.
